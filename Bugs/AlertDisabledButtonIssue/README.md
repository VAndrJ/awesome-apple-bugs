## Problem


Alert issue: `.disabled` modifier does not work correctly up to iOS 18.


## Environment


- Xcode 15-16.
- iOS ...17.
- iPadOS ...17.
- Swift 5/6.


## Solution / Workaround


Use Representable for alerts on iOS below 18.
