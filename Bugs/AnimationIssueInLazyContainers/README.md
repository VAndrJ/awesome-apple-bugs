## Problem


Animation stops working in Lazy containers (Stacks, Grids) when navigating to the next screen and returning.


## Environment


- Xcode 16.0-16.3 (current; check on future versions).
- iOS 18.2.
- Swift 5/6.


## Solution / Workaround


Set the `id` and reset in `onAppear`.
