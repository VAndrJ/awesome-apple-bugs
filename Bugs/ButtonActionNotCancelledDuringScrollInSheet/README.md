## Problem


The button action is not canceled during scrolling in sheet and is executed after scrolling completes.


## Environment


- Xcode 16.0-16.2 (current; check on future versions).
- iOS 18.0-18.2 (current; check on future versions).
- Swift 5/6.


## Solution / Workaround


Build in Xcode 15.4.
