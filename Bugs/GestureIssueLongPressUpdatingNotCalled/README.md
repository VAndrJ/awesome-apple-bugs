## Problem


Gesture issue: `.updating` block is not called on `LongPressGesture`.


## Environment


- Xcode 16.0-16.2 (current; check on future versions).
- iOS 18.0-18.2 (current; check on future versions).
- Swift 5/6.


## Solution / Workaround


- Build with Xcode 15.4.
- Use different gestures.
