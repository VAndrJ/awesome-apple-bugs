## Problem


Navigation title is displayed in quotation marks `" "` if a whitespace is specified as the title.


## Environment


- Xcode 16.
- iOS 18.0+.
- iPadOS 18.0+.
- Swift 5/6.


## Solution / Workaround


Set `title = nil` programmatically.
